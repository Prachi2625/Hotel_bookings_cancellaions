# 🏨 Hotel Bookings Cancellation Dashboard

## 📊 Project Overview
This project analyzes hotel booking cancellations using Excel dashboards.
The dashboard provides monthly insights, cancellation trends, and hotel type comparisons.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Charts (Bar, Line, Pie)
- Dashboard Design

---

## 📁 Dataset

## 📈 Key Insights

- City Hotels show higher cancellation rates compared to Resort Hotels.
- Peak cancellations occur during high-demand months.
- Family bookings fluctuate seasonally.
- Cancellation rate impacts revenue planning significantly.
